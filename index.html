HTML 5

Web

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear la estructura y el contenido de las páginas web. Es el lenguaje base de la World Wide Web y se utiliza para definir la estructura y el significado de los elementos de una página web, como encabezados, párrafos, enlaces, imágenes, formularios, tablas, etc.

Usos

En resumen, HTML es el lenguaje fundamental para crear páginas web, proporcionando la estructura y el contenido básico de una página que puede ser aplicado en:

  • Creación de sitios web.
  • Desarrollo de aplicaciones web.
  • Construcción de formularios en línea.
  • Publicación de contenido en línea.

Historial

HTML lo aprendí a fondo en la formación de Oracle One de Alura Latam, en sus cursos de HTML.

CSS

Diseño Web

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para definir la presentación y el diseño de un documento HTML. Permite controlar el aspecto visual de un sitio web, incluyendo colores, fuentes, márgenes y disposición de los elementos en la página.

Usos

En resumen, Se utiliza en conjunto con HTML para crear páginas web con estilos.

  • Dar formato y estilo a páginas web.
  • Crear diseños responsivos y adaptativos.
  • Animaciones y efectos visuales.
  • Personalización de la apariencia de los elementos en una página.

JavaScript

Programación

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza principalmente en el desarrollo web. Es un lenguaje de scripting del lado del cliente, lo que significa que se ejecuta en el navegador del usuario final.

Usos

En resumen, JavaScript es un elemento fundamental que permite interactuar con el contenido de una página web, modificar su estructura y comportamiento de manera dinamica.

  • Validación de formularios.
  • Manipulación y modificación de elementos HTML.
  • Interacción con el usuario, como eventos y animaciones.
  • Desarrollo de aplicaciones web en el lado del cliente.

Laravel

Framework

¿Qué es Laravel?

Laravel es un framework de desarrollo de aplicaciones web en PHP. Proporciona una estructura robusta y una amplia gama de funcionalidades para agilizar el desarrollo.

Usos

  • Desarrollo de aplicaciones web escalables y de alto rendimiento.

  • Gestión de bases de datos y consultas.
  • Implementación de autenticación y autorización.

Vue JS

Framework

¿Qué es JavaScript?

Vue.js es un framework de JavaScript utilizado para construir interfaces de usuario interactivas. Es conocido por su enfoque progresivo, lo que significa que se puede utilizar de forma incremental en un proyecto existente.

Usos

  • Desarrollo de aplicaciones web interactivas y de una sola página.
  • Construcción de componentes reutilizables.
  • Actualización de la vista de una aplicación de manera reactiva y eficiente.

Progressive Web App

Multiplataforma

¿Qué es PWA?

Las Progressive Web Apps (Aplicaciones Web Progresivas) son aplicaciones web que se comportan como aplicaciones nativas en dispositivos móviles y escritorio.

Usos

Proporcionan una experiencia similar a la de una aplicación nativa, como notificaciones push, acceso fuera de línea y la capacidad de instalación en el dispositivo.

  • Desarrollo de aplicaciones móviles multiplataforma.
  • Creación de experiencias de usuario fuera de línea.
  • Mejora de la accesibilidad y rendimiento de una aplicación web.
  • Fácil distribución de aplicaciones

Aplicación de Check List

Proyecto de Estadías

Descripción del Proyecto

Este proyecto es una aplicación desarrollada en Vue (usando Bootstrap para los estilos) para el registro de las tareas diarias del área de sistemas en el hotel Ocean Riviera Paradise, adicionalmente se creó un módulo para la administración de los check list de los 7 hoteles pertenecientes a la cadena Ocean en el sistema del hotel el cual está desarrollado con el framework Laravel.

Funciones Principales

Este proyecto cuenta con las siguientes funcionalidades:

  • Conexión entre ambos sistemas mediante soporte API de Laravel.

    • Aplicación

  • Login a la aplicación usando los usuarios del sistema del hotel.
  • Ver un resumen de las actividades del hotel al que pertenece el usuario.
  • Llenar todos los ítems del check list del hotel seleccionado, evidencia incluida.
  • Editar el check list realizado del día actual.
  • Ver los check list realizados, mediante un filtro de fechas.

    • Modulo de administración

  • Crear check list para un hotel.
  • Crear items de un check list.
  • Editar y eliminar ítems o check list.
  • Ver los registros de checklist realizados mediante filtros.
  • Apartado de estadísticas de los registros realizados.

Experiencias

Realizar este proyecto fue un desafio dada la complejidad que se presentó al momento de conectar ambos sistemas, puse todo mi empeño y esfuerzo para mejorar mis habilidades y conocimientos para así poder entregar un producto de calidad que al momento de concluir mis estadías quedó 100% terminado y listo para poner en producción. Me hubiera gustado realizar esa etapa sin embargo el tiempo no fue suficiente.

Dificultades

Presenté algunas complicaciones a lo largo del desarrollo de la aplicacion. Al principio fue un reto entender la estructura y funcionamiento del sistema del hotel pues hacia uso de varios elementos tanto de Laravel como de Vue que no habia visto antes, sin embargo no me tomó tiempo decifrarlo y asi poder empezar a desarrollar el proyecto. Hubo confusiones en el modelo de datos pero fueron solucionadas con rápidez. En la aplicación el reto más complicado fue el llenado del check list pues fue solicitado el poder llenar y enviar todos los items en un solo formulario, tanto en la logica como en lo visual fue dificil de realizar pero investigando y haciendo pruebas logré resolverlo.

Easy Access

Primeramente fue desarrollado en laravel, actualmente se han realizado modificaciones al proyecto y se encuentra en etapa de diseño pero se planea desarrollar haciendo uso de Vue ó React para el FrontEnd y Laravel para el BackEnd.

  • Realizar Justificaciones.

Currículum PWA

Proyecto Escolar

Descripción del Proyecto

Este currículum es una actividad el la matería de desarrollo de software multiplataforma, utiliza la tecnología de las Progressive Web Apps para poder instalarse en el dispositivo lo que permite que sea accesible a cualquier dispositivo ya sea movíl o de escritorio.

Funciones Principales

Este currículum cuenta con las siguientes funcionalidades:

  • Uso de fetch para obtener datos de un JSON.
  • Inserción de contenido html de forma dinámica con JS.
  • Uso de service worker.
  • Instalación de la aplicación.
  • Uso del cache del dispositivo.
  • Carga de la aplicación sin Conexión a internet.

Experiencia

Realizar este proyecto me permitió conocer alternativas para el desarrollo en dispositivos movíles pues haciendo uso del Manifest y el Service worker se puede crear un producto que parece desarrollado de manera nativa pero que no importa el dispositivo o el sistema la mayoría puede hacer uso de esta aplicación.

Dificultades

Lo complicado de este proyecto fue entender el funcionamiento de una PWA y el como se podria llevar a cabo además de la obtencion de datos, si embargo investigando lo suficiente se logro desarrollarlo completamente y sobre todo que es funcional.
